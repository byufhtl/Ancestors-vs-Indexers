User Guide

Getting Started

Begin by creating or loading a new document from the File menu.
You can always change the name and author information later. Keep scripts all named different things, if you can.

Filling it in

Once a file has been created, fill in the speaker and the message for a single character's statement.
A message should be all that you want a player to say in a single frame of text. If what is being said will require two frames of speach, use the "Next" button to create another dialogue box and continue from there.
Try to keep messages to 2 sentences.

Press "Next" to create another dialogue box.

Use the "Previous" button to fill in a frame before the current frame of reference.

Use the "Delete" button to delete a frame of dialogue. YOU CAN'T UNDO THIS.

Events

If something needs to happen (e.g. another player walks up to you and starts talking), let us know by including a description of what you would like to have happen. If something needs to be added to the player's inventory, an objective should be marked as filled, or an objective should be added to the game at the end of a dialogue, the event should be logged in the event box of the last script in the sequence.

Saving & Loading

Once a script has been completed, you may save it to your machine via the File menu.
Be sure to update the Author and Script name fields in the area above.
You may upload a script you have previously worked on using the load feature in the file menu.
If you refresh your page without saving you will lose your work, so save often.

Submission

Once you have completed a script and are ready for us to plug it into the game, email us the .scr file you saved to your computer.
